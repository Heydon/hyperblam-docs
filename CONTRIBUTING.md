# Contributing

If you are looking to contribute to **HYPERBLAM** itself, that repository is hosted with [git.gay](https://git.gay/heydon/hyperblam) and has its own [contribution guide](https://git.gay/heydon/hyperblam/src/branch/main/CONTRIBUTING.md).

Since this repository is for documentation, **clarity** and **completeness** are the main things to get right. If you see something that doesn’t make sense or isn't explained, please write up an issue.

The neatness or cleverness of the _code_ here is of less importance than performance. It's just a cobbled together [Eleventy](https://www.11ty.dev/docs/) site. If you experience performance issues, please write an issue describing the problem and—preferably—also include data from dev tools (or whatever diagnostic tool you are using).
